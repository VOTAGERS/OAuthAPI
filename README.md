## Pre-Requisite
-	Install dependencies
	> npm install
- Synchronize model schema
	> npx prisma generate \
	> npx prisma db push


## Usage & Method

- GET
> /users : fetch user data \
> /roles : fetch role data \
> /dept : fetch department data

- POST
> POST : insert data \
> PATCH : update data
